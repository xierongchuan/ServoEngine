# Setup Virtual Environment

1.  **Create venv:**
    ```bash
    python3 -m venv venv
    ```

2.  **Activate:**
    ```bash
    source venv/bin/activate
    ```

3.  **Install dependencies:**
    ```bash
    pip install -r requirements.txt
    ```

4.  **Verify:**
    ```bash
    which python
    # Should point to .../venv/bin/python
    ```
